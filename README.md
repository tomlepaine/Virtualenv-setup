# Virtualenv-setup
Some helper files to setup virtualenv the way I like.

## Usage
To use, run:
```bash
cd $VIRTUAL_ENV/bin/
wget https://raw.githubusercontent.com/tomlepaine/Virtualenv-setup/master/setup.sh
chmod 777 setup.sh
./setup.sh
```

To use pre-commit hooks. In a git repo run:
```bash
pre-commit-setup
```

## Requirements
- git
- jupyter
- pre-commit
